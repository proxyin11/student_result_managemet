 # Student Result Management System

## About the Project

This is a simple Student Result Management System developed using Python. I created this project to practice basic Python programming concepts such as dictionaries, loops, conditional statements, and user input.

The application is menu-driven and runs in the command line. It allows users to add student records, view all students, and check whether a student has passed or failed based on their marks.

## Features

* Add a new student with marks
* View all student records
* Check whether a student has passed or failed
* Simple and easy-to-use command-line interface

## Technologies Used

* Python 3
